Day 17 - Cavalcade of Rollovers

#1 - Pure CSS Image Rollovers (needed in P3)

The trick: use a:hover and a:active to change the background-image property of an <a> tag.

#2 - Simple JavaScript Rollovers (optional)

The trick: use inline event handlers and JavaScript programming to change the value of the src attribute of an <img> tag.

Note: Using inline JavaScript event handlers is discouraged for the same reason that inline CSS is - they both muddle up your HTML content. We will demo a "cleaner" way to do this on Day 19.

#3 - "Disjoint" JavaScript Rollovers (optional)

Changing the src attribute of a different element is a neat effect that you can't do with CSS alone. The technique we're showing here is a little "old school" though - we'll show you a better one on Day 19.
