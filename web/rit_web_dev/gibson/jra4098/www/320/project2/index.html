About the Repository

The "Hello, world!" repository showcases the basic syntax of a variety of programming languages. For each language listed, simple examples of writing output and reading input will be shown.

The repository is not intended to be a one-stop-shop for learners; you will not find extensive how-to guides here, and you will not learn any new languages by browsing these pages. Instead, the focus of the repository is on quality of code samples and (secondary to the first objective) breadth of languages.
