"Hello, world!" in C

/*-==========================================================================-*\ Author(s): Jeremy Audet First Created: 10/24/2011 License: Creative Commons BY-SA http://creativecommons.org/licenses/by-sa/3.0/ \*-==========================================================================-*/ #include "stdio.h" int main(void) { printf("Hello, world!\n"); return 0; }

The \n character is an escape code that means "start a new line".

Echo in C

/*-==========================================================================-*\ Author(s): Jeremy Audet First Created: 10/25/2011 License: Creative Commons BY-SA http://creativecommons.org/licenses/by-sa/3.0/ Thanks to: * Beej's Guide to C http://beej.us/guide/bgc/output/html/multipage/putc.html I had forgotten how easy it is to write out characters using putchar(). \*-==========================================================================-*/ #include <stdio.h> #include <stdlib.h> int main(void) { char input; // For buffering input. printf("Enter input, and it will be immediately echoed back.\n" "Press CTRL-D or CTRL-C to quit.\n"); while((input = getchar()) != EOF) { putchar(input); } putchar('\n'); return 0; }

C does not care about whitespace. This allows the printf() statement to be split over two lines.
